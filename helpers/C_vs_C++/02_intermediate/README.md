# Intermediate Level — C vs C++

These programs demonstrate OOP concepts and manual data structure implementations.

| # | Topic | Key Difference |
|---|-------|---------------|
| 1 | Structs vs Classes | C structs (data only) vs C++ classes (encapsulation) |
| 2 | Dynamic Memory | `malloc/realloc/free` vs `new/delete` + smart pointers |
| 3 | File I/O | `fopen/fread/fwrite` vs streams (`fstream`) |
| 4 | Linked List | Manual pointer management vs class-based + STL `list` |
| 5 | Stack | Array/linked-list based vs `std::stack` |
| 6 | Queue | Circular buffer vs `std::queue` + `std::deque` |

## Summary

At the intermediate level, C++ introduces **encapsulation** (classes), **RAII** (automatic resource management), and **STL containers** that replace hundreds of lines of manual C code with a few well-tested library calls.
