# Basic Level — C vs C++

These programs cover fundamental syntax and feature differences.

| # | Topic | Key Difference |
|---|-------|---------------|
| 1 | Hello World | `printf` vs `cout` |
| 2 | Variables & I/O | `scanf` vs `cin`, type safety |
| 3 | Arrays | Fixed arrays vs `std::array` |
| 4 | Strings | `char[]` + `string.h` vs `std::string` |
| 5 | Functions | No overloading in C vs overloading + default args in C++ |
| 6 | Pointers | Raw pointers vs references + smart pointers |

## Summary

At the basic level, C++ provides **syntactic sugar** and **type safety** over C. The logic remains the same, but C++ reduces boilerplate and catches more errors at compile time.
