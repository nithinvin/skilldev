.container { display: flex; flex-direction: row; justify-content: flex-start; align-items: stretch; flex-wrap: nowrap; gap: 10px; }
Try this: Set justify-content to "space-between" and then add more items. Notice how the space is distributed EVENLY between items, with first and last items flush to edges.

Quick Reference

Property Applies To What It Does
display: flex Container Enables flex layout on children
flex-direction Container Main axis direction (row/column)
justify-content Container Distribute space along MAIN axis
align-items Container Align items along CROSS axis
flex-wrap Container Allow items to wrap to next line
gap Container Space between items (no margin hacks!)
flex-grow Item How much extra space to absorb (0=none)
flex-shrink Item How much to shrink if space is tight
align-self Item Override align-items for one item
