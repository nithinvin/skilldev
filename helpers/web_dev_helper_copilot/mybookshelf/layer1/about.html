What is this?

MyBookShelf is a learning project built layer by layer, from raw TCP sockets to a full-stack deployed application.

The Layers

  1. Layer 0 — Linux, networking, raw sockets (completed!)
  2. Layer 1 — HTML, CSS, JavaScript (you are here)
  3. Layer 2 — Backend server + PostgreSQL database
  4. Layer 3 — REST API, authentication, authorization
  5. Layer 4 — Docker, CI/CD, monitoring
  6. Layer 5 — Cloud deployment, microservices
  7. Layer 6 — Cryptography & security
  8. Layer 7 — Machine learning, LLMs, MCP
  9. Layer 8 — Data analytics & dashboards

Technology Stack

Frontend
HTML, CSS, JavaScript (no frameworks — understand the fundamentals first)
Backend
Python (Flask/FastAPI) — because you already know Python
Database
PostgreSQL (relational) + Redis (cache)
Infrastructure
Docker, nginx, GitHub Actions CI/CD
Deployed on
Hetzner Cloud (Ubuntu VM)

Design Philosophy

"I hear and I forget. I see and I remember. I do and I understand."

— Confucius (paraphrased)

Every layer builds something real and runnable. No theoretical exercises without code. No code without understanding.
